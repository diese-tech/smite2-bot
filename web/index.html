FAQ

Prototype boundaries, clearly marked.

Does this add real Discord OAuth?

No. Connect Discord is visual-only in this prototype. Real OAuth is tracked in TODO.md.

Is command config saved anywhere?

No. The builder is a product demo. Bot-side command configuration storage and execution are future work.

Does the website call the bot backend?

No. This is static HTML, CSS, and JavaScript. A shared API layer is planned later.

Where do production graphics go?

The page uses named asset slots for god-card, item-card, role-icon, dashboard-hero, and background-texture.
